缓动
  • 1.为运动确定一个比例系数，大于0、小于1
  • 2.确定目标点
  • 3.计算物体与目标点的距离
  • 4.计算速度，速度=距离*比例系数
  • 5.用当前的位置加上速度来计算新的位置
  • 6.重复3到5步，直到物体到达目标
距离
var dx = targetX - ball.x;
var dy = targetY - ball.y;

速度
var vx = dx * easing;
var vy = dy * easing;
ball.x += vx
ball.y += vy


简化=>


vx = (targetX - ball.x)*easing
vy = (targetY - ball.y)*easing
ball.x += vx
ball.y += vy


再简化=>


ball.x +=  (targetX - ball.x) * easing
ball.y +=  (targetY - ball.y) * easing

缓动系数 速度除以距离可以得到这个系数，小于1、且大于0
越接近0，运动速度越慢
越接近1，运动速度越快
